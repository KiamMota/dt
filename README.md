# dt

A small Go utility that moves files and directories into a target folder.

`dt` automatically creates the destination directory if it does not exist, making it easier to organize project files without manually creating folders and typing long `mv` commands.

## Features

- Move files and directories to a destination folder
- Automatically create the destination directory
- Simple command-line interface
- Written in Go
- Cross-platform

## Installation

Clone the repository:

```bash
git clone https://github.com/your-user/dt.git
cd dt
go install
